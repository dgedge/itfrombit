# Documentation

- `script_manifest.csv` is generated by `tools/import_canon_scripts.py`.
- `canon_material/` contains selected orientation notes and derivation diagrams
  mirrored from the working canon tree.
- [`flawed_analysis/A_case_study_in_Science.md`](flawed_analysis/A_case_study_in_Science.md)
  is the revised AI-assisted-science case study, with linked image assets in
  `flawed_analysis/assets/`.
- Topic-level README files live under `scripts/<topic>/README.md`.
- The root `README.md` describes the current repository layout and curation
  standard.

The manifest is the best starting point for finding which script supports a
given canon or paper claim.
